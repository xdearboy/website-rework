# Итоги 2025

---

## в роли Разработчика

### Что делал

- Писал ботов для Discord и свои утилиты  
- Делал небольшие проекты на Python и TypeScript  
- Настраивал серверы, ковырял Linux, Docker,   
- Учился DevOps и разбираться с деплоем

### Главные проекты

**mfetch** — утилита для отображения информации об ОЗУ.  
- Делал: сбор данных, вывод в консоль, кастомизация внешнего вида
- Чему научился: Rust, работа с SPD, оформление CLI

**pterodactyl-crasher** — инструмент для тестирования серверов Pterodactyl.  
- Делал: понимание API, работу с сетью, тестирование нагрузок  
- Чему научился: API-запросы, структура проектов, основы сетевой безопасности (для тестов, легально!)

**dec64** — экспериментальный проект на Python.  
- Делал: экспериментировал с разработкой библиотек для Python  
- Чему научился: логика, оптимизация, работа с base64

### Чему научился

- Структурировать код, чтобы не путаться через неделю  
- Читать документацию и использовать её по делу    
- Не бояться больших задач
