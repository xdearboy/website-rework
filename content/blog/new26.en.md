# 2025 in Review

## As a Developer

### What I did

- Wrote bots for Discord and my own utilities
- Built small projects in Python and TypeScript
- Set up servers, poked around in Linux and Docker
- Learned DevOps and how to deal with deployments

### Main projects

**mfetch** - a utility for displaying RAM information.
- What I did: data collection, console output, customizing the appearance
- What I learned: Rust, working with SPD, CLI design

**pterodactyl-crasher** - a tool for testing Pterodactyl servers.
- What I did: understanding the API, working with networking, load testing
- What I learned: API requests, project structure, basics of network security (for testing, legally!)

**dec64** - an experimental Python project.
- What I did: experimented with building libraries for Python
- What I learned: logic, optimization, working with base64

### What I learned

- How to structure code so I don't get lost a week later
- How to read documentation and actually use it
- Not to be afraid of big tasks
