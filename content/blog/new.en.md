## A Markdown-powered blog system

Built a blog for the site with Markdown support. It's simple - you write in `.md` files, and the system automatically turns them into a web page with rendered Markdown.

### What it can do:

- Write posts using Markdown.
- Automatic conversion of `.md` into a web page with rendered Markdown.
- Easy to add new content.
